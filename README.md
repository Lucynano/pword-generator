# PWORD-GENERATOR

A **Random Password Generator** in Python; allow you to choose the password length, to mix the types of characters like uppercase letters, lowercase letters, digits and special characters


## Features
1. Generate a random password

2. Choose the password length (5 ≤ length ≤ 128)

3. Choose the types of characters:
   - 'QWERTYUIOPASDFGHJKLZXCVBNM'
   - 'qwertyuiopasdfghjklzxcvbnm'
   - '1234567890'
   - '!@#$%^&*'


## Steps to follow
1. Clone the repository
   
   ```bash
   git clone https://github.com/Lucynano/pword-generator.git
   ```

2. Go to the project directory

   ```bash
   cd pword-generator
   ```

 3. Run the script

  ```bash
  python pword_generator.py
  ```
